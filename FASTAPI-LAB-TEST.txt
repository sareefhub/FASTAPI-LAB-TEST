Step ฺBy Step

1. สร้าง venv 

```
py -m venv venv
```

2. เปิดใช้งาน venv

```
.\venv\Scripts\activate
```

3. ติดตั้ง FastAPI เฉพาะใน venv นี้

```
pip install "fastapi[standard]"
pip install pydantic
pip install sqlalchemy
```

4. รันเซิร์ฟเวอร์ FastAPI ในโหมดพัฒนา (development)

```
uvicorn main:app --reload
```

ตั้งค่า VS Code ให้ใช้ venv เป็น Python Interpreter

1.กด Ctrl+Shift+P (หรือ F1) เพื่อเปิด Command Palette

2.พิมพ์

```
Python: Select Interpreter
```

3.เลือก interpreter ที่อยู่ใน path แบบนี้

```
.\venv\Scripts\python.exe
```

อ่านฐานข้อมูล

```
python read_db.py
```
